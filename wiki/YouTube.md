# YouTube

Video publishing connector for **ScriptLabs Studios**.

## Role

Highest-leverage distribution surface for technical credibility and audience growth. Keeps video output visible in the operations graph and automatable via [[ScripLabStudio8]].

## Typical workflows

- Publish demo recordings (5-min cold demos, 30-min scoped demos per [[Playbooks/Demo Playbook]])
- Publish architecture walkthroughs and studio build updates
- Publish founder content aligned with [[WORLDVIEW]] trend captures
- Cross-post clips to [[Discord]], [[Telegram]], [[WhatsApp]]

## Connection notes

- Auth: YouTube Data API v3 OAuth credentials, stored outside vault
- Automation: upload, schedule, metadata (title, description, tags, thumbnail)
