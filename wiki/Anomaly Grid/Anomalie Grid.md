# Anomalie Grid

Flagship AI-security platform node for Intellusia Studios.

## Description

Anomalie Grid is the flagship AI-security platform concept in the vault.
It serves as the umbrella product node that connects the major product components, workflow nodes, and technical story around Intellusia Studios.

## Why it matters

This is the main commercial and architectural center of the company map.

### Platform components
- [[Sentinel Swarm]] — edge collectors (data in)
- [[Nebuchadnezzar]] — Morpheus + Triton inference engine (data processed)
- [[The Construct]] — honeynet / decoy intelligence (attacker data in)
- [[The Oracle]] — agentic reasoning layer (predictions out)
- [[Zion]] — tenant-facing portal (human sees it)
- [[The Architect]] — admin control plane (operator manages it)
- [[Red Pill Onboarding]] — customer entry path

### Sibling product
- [[Anomaly Grid CLI]] — solo-operator Rust CLI (defensive, local-first, no SaaS)

### Knowledge layer
- [[Kofi Cyber Synthesis]] — analyst-side SOC/cyber fundamentals; MITRE, Kill Chain, Diamond Model, Pyramid of Pain; defines the buyer language and detection-design vocabulary

- [[Intellusia Studios]]
