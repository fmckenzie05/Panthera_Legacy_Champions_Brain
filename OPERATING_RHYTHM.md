# OPERATING_RHYTHM

The default cadence and workflow patterns for Intellusia Studios. Read this so the vault can be operated without the founder narrating each step.

Related:
- [[OPERATOR_PROFILE]]
- [[OpenClaw]]
- [[WORLD_VIEW]] (trend-scanning cadence lives there too)

## Daily

- Open [[HOME]], skim [[MEMORY]], glance at active strategy notes ([[FIRST_POPULATION_NOTE]], [[OPENCLAW_OPERATIONS_AND_CONNECTOR_STRATEGY]], [[GOVERNMENT_CONTRACTING_PIPELINE_STRATEGY]]).
- Pick the single most leveraged task aligned with [[OPERATOR_PROFILE]] priorities.
- Write any new fact, decision, or surprise into the relevant node. Do not let knowledge live only in chat history.

## Weekly

- Review [[WORLD_VIEW]] sources. Capture anything that should change the build or the pitch.
- Update [[MEMORY]] if anything material changed (status of [[Anomalie Grid]], new node, new client signal).
- One outbound action: a post, a demo recording, an outreach batch, or a capability statement update.

## Monthly

- Audit the vault for stale notes (status: drafts that have shipped, projects that died, links that rotted).
- Re-read [[WORLDVIEW]] and [[DECISION_PRINCIPLES]]. Edit if a belief has changed.
- Rebalance priorities in [[OPERATOR_PROFILE]] if the company stage has shifted.

## When something new happens

| Event | Where it lives |
| --- | --- |
| New idea | New idea note, link from [[HOME]] |
| Meeting with substance | New meeting note using template |
| Decision that affects strategy | Update relevant strategy note + [[MEMORY]] |
| New external connector | Stub note + `Rel - OpenClaw to X` + update [[INTELLUSIA_GRAPH]] |
| New client touchpoint | Update [[CLIENT_CONNECTION]] |
| New trend to track | Add to [[WORLD_VIEW]] with a source |

## What "done" looks like for a working session

A session ends with at least one of:

- A node added or sharpened.
- A decision recorded with its reasoning.
- A shippable artifact produced (post, page, slide, email).
- An automation wired through [[OpenClaw]].

If a session ends with none of those, it was research — log what was learned in the relevant node so it compounds.
